Como Configurar a API Key

Passo a passo:
  1. Obtenha sua API Key:
    • Acesse: Google AI Studio
    • Faça login com sua conta Google
    • Clique em "Create API Key" ou "Get API Key"
    • Copie a chave gerada
  2. Configure no .env:
    GEMINI_API_KEY=AIzaSyXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
  3. Reinicie o servidor:
    python manage.py runserver
  4. Teste novamente:

    Recarregue esta página ou acesse o chatbot para verificar.

Teste via Linha de Comando

Você também pode testar a API key via terminal Django:

# Teste básico
python manage.py test_gemini

# Teste detalhado
python manage.py test_gemini --detailed

# Teste com mensagem customizada
python manage.py test_gemini --message "Recomende um romance brasileiro"
